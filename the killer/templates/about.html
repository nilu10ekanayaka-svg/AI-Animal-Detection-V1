සජීවී කැමරා නිරීක්ෂණය

HD ගුණාත්මක කැමරාවක් භාවිතා කරමින් වත්තේ සියලුම ක්‍රියාකාරකම් සජීවීව නිරීක්ෂණය කරයි. සතුන් අනාවරණය වූ විට ස්වයංක්‍රීයව ඇලම් සහ SMS යවයි.

AI අනාවරණ පද්ධතිය

කෘතිම බුද්ධිය භාවිතා කරමින් සතුන් සහ මිනිසුන් අතර වෙනස හඳුනා ගනී. සතුන්ගේ නම් සහ ප්‍රමාණය ස්වයංක්‍රීයව අනාවරණය කරයි.

SMS ඇලම් පද්ධතිය

Twilio සේවාව භාවිතා කරමින් සතුන් ඇතුළු වූ විට කර්මිකාරයාට SMS පණිවිඩයක් යවයි. ජාත්‍යන්තර SMS සේවාව සමඟ සම්බන්ධ වේ.

ශබ්ද ඇලම් පද්ධතිය

සතුන් අනාවරණය වූ විට ස්වයංක්‍රීයව ශබ්ද ඇලම් ආරම්භ කරයි. ඔබේම ශබ්ද ගොනු උඩුගත කර භාවිතා කළ හැකිය.

සිදුවීම් ඉතිහාසය

සියලුම අනාවරණ සිදුවීම් CSV ගොනුවක් ලෙස සුරක්ෂිත කරයි. දිනය, වේලාව සහ සිදුවීම් විස්තර සමඟ සම්පූර්ණ වාර්තාවක් ලබා දේ.

වෙබ් අතුරුමුහුණත

ඕනෑම උපාංගයකින් ප්‍රවේශ විය හැකි responsive වෙබ් අතුරුමුහුණත. දුරකථන, ටැබ්ලට් හෝ පරිගණකයකින් පහසුවෙන් පාලනය කළ හැකිය.

තාක්ෂණික විස්තර

Python & OpenCV

Python programming language සහ OpenCV computer vision library භාවිතා කරයි.

Flask web framework භාවිතා කරමින් RESTful API සහ web interface සපයයි.

Twilio SMS

Twilio cloud communications platform භාවිතා කරමින් SMS සේවා සපයයි.
