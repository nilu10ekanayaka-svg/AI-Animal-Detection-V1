🐾 AI Animal Detection System

🚀 Features

  • Real-time animal detection using AI/ML
  • Web-based dashboard for monitoring
  • SMS alert system for notifications
  • Event logging and tracking
  • Alarm system with audio alerts
  • Admin panel for configuration

📋 Quick Start

# Install dependencies pip install -r requirements.txt # Run the application python app.py # Access the web interface # Open http://localhost:5000 in your browser

📁 Project Structure

AI-animal-detection/ ├── app.py # Main Flask application ├── detector.py # AI detection logic ├── alarm_system.py # Audio alert system ├── sms_system.py # SMS notification system ├── templates/ # HTML templates ├── static/ # CSS, JS, images ├── requirements.txt # Python dependencies └── README.md # Documentation

🔧 Setup Instructions

  1. Clone this repository
  2. Install Python dependencies: pip install -r requirements.txt
  3. Configure your settings in config.json
  4. Set up SMS credentials (optional)
  5. Run: python app.py
  6. Open browser to http://localhost:5000
